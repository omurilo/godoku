A blazing fast static site generator for documentation and API references, written in Go.

- ⚡️ Instant builds
- 🦾 Markdown + OpenAPI + Mermaid
- 🎨 Customizable themes
- 🌙 Light/dark mode
- 🔍 Full-text search
- 🧩 API playgrounds
- 📝 Edit this page links
- 🏷️ SEO & social meta
- 📦 Single binary, zero dependencies

## Why Godoku?

Godoku is designed for technical docs, API references, and developer portals. It combines the best of Markdown, OpenAPI, and modern JS/Go tooling for a delightful authoring and reading experience.

- **Markdown-first:** Write docs in plain Markdown, with support for code blocks, admonitions, and more.
- **API-native:** Render OpenAPI specs as beautiful, interactive API reference docs.
- **Customizable:** Brand, theme, and extend your docs with custom CSS/JS.
- **Fast:** Instant rebuilds, even for large docs.
